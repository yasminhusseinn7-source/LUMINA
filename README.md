# LUMINA — Art & Light Exhibition

A responsive event landing page for a fictional art and light exhibition in Alexandria, Egypt.

## Built With
- HTML
- CSS (Flexbox, Media Queries)
- JavaScript (DOM Manipulation)

## Features
- Hero section with event details
- Reservation form
- Confirmation message on submit
- Mobile responsive
